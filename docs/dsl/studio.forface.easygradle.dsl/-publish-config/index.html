PublishConfig

class PublishConfig

Holds params for publication. Each parameter can be set

  • programmatically
  • in gradle.properties
  • as Environment variable properties names in gradle.properties reflect the field name of the variable and environment variables reflect the property name name as uppercase snake_case where not specified differently. See gradle.properties.template for more examples
