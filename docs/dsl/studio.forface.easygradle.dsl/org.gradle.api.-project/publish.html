publish

fun Project.publish(baseBlock: PublishConfigBuilder? = null, artifact: String? = null, block: PublishConfigBuilder = {}): Unit

Apply publish script to the receiver Project

Params for PublishConfig can be set:

  • programmatically
  • in gradle.properties
  • in Environment variable

Parameters

artifact - Optional PublishConfig.artifact for the PublishConfig, this is useful when we have a stored common PublishConfig for the project and we want to apply it for a single module

baseBlock - Optional Lambda previously created by publishConfig for have a base setup for PublishConfig

block - Lambda for setup PublishConfig
